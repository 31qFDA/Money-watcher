ОПИСАНИЕ

Money Watcher - это десктопное приложение для учёта личных финансов, постановки финансовых целей и анализа доходов/расходов. Приложение создано с использованием Python и Tkinter.

ФУНКЦИОНАЛ

Управление финансами
- Отслеживание доходов и расходов с привязкой к датам
- Категоризация операций (доход/расход)
- Просмотр баланса в реальном времени
- Поддержка множественных валют (₽, $, €, £, ¥)

Финансовые цели
- Постановка целей с указанием целевой суммы
- Отслеживание прогресса с помощью прогресс-баров
- Возможность установки крайнего срока для целей
- Пополнение целей с автоматическим учётом в расходах
- Возврат накопленных средств при удалении цели

Визуализация и аналитика
- Круговая диаграмма доходов и расходов за месяц
- Карточки с финансовой сводкой (баланс, доходы, расходы)
- Таблицы с историей операций
- Сортировка данных по различным параметрам

Навигация по датам
- Выбор месяца для просмотра статистики
- Возможность ручного ввода даты
- Выпадающий список с доступными месяцами

Хранение данных
- Сохранение данных в JSON формате
- Создание резервных копий
- Смена файла данных
- Массовое удаление операций и целей
- Возможность полного сброса данных

ИНТЕРФЕЙС

Главная страница

Главная страница предоставляет быстрый доступ ко всей ключевой информации:
- В верхней чатси экрана расположены кнопки для добавления операций и целей, переключение режима множественного удаления, выбор месяца
- Отображение текущего баланса, месячных доходов и расходов
- Круговая диаграмма распределения доходов и расходов за выбранный месяц
- Список активных целей с прогресс-барами и возможностью быстрого пополнения
- Таблица последних операций

Страница операций

Полноценная таблица со всеми операциями:
- Сортировка по ID, дате, названию, типу и сумме
- Редактирование выбранной операции
- Удаление отдельных или множественных операций

Страница целей

Карточки с целями, отображаемые в сетке:
- Индикатор прогресса выполнения
- Текущая и целевая сумма
- Крайний срок (если установлен)
- Возможность пополнения, редактирования и удаления целей

Страница настроек

- Переключение между светлой и тёмной темой
- Выбор предустановленной валюты
- Сохранение, резервное копирование, смена файла данных, полный сброс отображающихся данных

ОБРАБОТКА ДАННЫХ

Все данные хранятся в JSON-файле расположенном в папке с самим приложением. При создании резервной копии через настройки создаётся папка "Backups" с датой его создания в названии файла.
